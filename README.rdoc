= Project Copy

管理者でないユーザにもプロジェクトのコピーを使えるようにできます。

一応、Redmine1.x、Redmine2.xに対応しています。
Redmine1.4.1/Redmine2.0.3をFireFox13.0.1で動作確認。

== インストール

1. RAILS_ROOT/vendor/pluginsに配置する。
   Redmine2.xの場合はRAILS_ROOT/pluginsに配置する。

2. Redmineを再起動する。


== アンインストール

1. RAILS_ROOT/vendor/pluginsから削除する。
   Redmine2.xの場合はRAILS_ROOT/pluginsから削除する。

2. Redmineを再起動する。
